# React-Redux-Shopping-Cart

Live Demo :- https://duha2001.github.io/Shopping-Cart/

### Instructions :-

- Clone the repository
- Go to project folder, Open terminal and run **`npm install`**
- then run **`npm start`**
- Open browser and go to **`http://localhost:3000`**
